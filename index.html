Technical Architecture

Stack Overview

  • CommonsRosca: core contract managing rounds, payouts, and commons fee accrual
  • MockERC20: deterministic stablecoin for local testing and demo clarity
  • MockYieldAdapter: plug-in yield strategy streaming LP returns to treasury
  • Frontend: Next.js + wagmi for wallet auth, state visualization, and admin flows
